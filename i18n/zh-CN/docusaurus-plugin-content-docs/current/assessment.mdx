# 自评估工具

import Assessment from '@site/src/components/FluencyModel/Assessment';

使用这个交互式工具评估您的团队在五个维度上的当前流畅度等级。该工具将：

1. **引导您完成每个维度** - 为每个维度选择当前等级
2. **保存您的进度** - 您的选择会自动保存在浏览器中
3. **提供改进建议** - 根据您的当前状态获得具体的下一步行动

## 使用方法

1. 对于每个维度，点击最能代表您团队当前状态的数字（1-5）
2. 如果不确定，请查看等级描述
3. 选择完所有维度后，点击"改进建议"查看个性化的改进路径
4. 使用"重置评估"重新开始

## 理解您的结果

您的评估结果显示：

- **当前状态**：您的团队在每个维度上的现状
- **下一级别**：每个维度的下一个目标
- **关键投入**：进入下一级别所需的具体行动

## 准确评估的技巧

- **诚实面对**：评估实际现状，而不是期望状态
- **考虑整个团队**：基于团队整体能力评估，而非个别优秀者
- **定期回顾**：每 3-6 个月重新评估以跟踪进度
- **聚焦单一维度**：不同维度处于不同等级是正常的
- **渐进式进步**：建议每次只提升一个等级

## 下一步行动

完成评估后：

1. **与团队分享** - 讨论结果并就优先级达成一致
2. **设定现实目标** - 首先选择 1-2 个维度进行重点改进
3. **制定行动计划** - 以关键投入为起点
4. **跟踪进度** - 设定里程碑并定期回顾
5. **持续迭代** - 根据实际效果调整方法

记住：目标不是立即在所有维度达到最高等级。专注于与团队需求和能力相符的可持续、渐进式改进。
