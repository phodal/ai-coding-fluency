# 交互式矩阵

import InteractiveMatrix from '@site/src/components/FluencyModel/InteractiveMatrix';

这个交互式矩阵让您可以详细探索 AI 编码流畅度模型。点击任何单元格查看该等级和维度组合的详细信息。

## 使用方法

- **悬停**在行或列上以高亮显示
- **点击**任何单元格查看详细信息
- **再次点击**或按 ✕ 按钮关闭详情面板

<InteractiveMatrix />

## 理解矩阵

矩阵展示了五个维度如何在五个流畅度等级中演进：

### 行（等级）
每一行代表一个流畅度等级，从顶部的等级 1（认识/意识唤醒）到底部的等级 5（智能体优先自治）。

### 列（维度）
每一列代表五个评估维度之一：
- **人机协作**：协作模式如何演进
- **生命周期覆盖度**：AI 覆盖多少开发阶段
- **工程化支撑**：基础设施成熟度
- **质量与治理**：质量保证机制
- **上下文工程**：AI 的上下文能力

### 单元格
每个单元格描述该维度在该等级的关键特征。点击单元格可查看：
- 特征标题
- 详细描述
- 与相邻等级的差异

## 关键洞察

- **非均匀进展**：团队不需要在所有维度上处于相同等级
- **依赖关系**：某些维度可能需要先进步，其他维度才能跟进
- **渐进步骤**：每个等级都建立在前一个等级之上
- **因地制宜**：正确的等级取决于团队的目标和约束
